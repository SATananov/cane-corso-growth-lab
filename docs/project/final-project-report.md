# Cane Corso Growth Geometry Lab — Product & ML Project Report

## 1. Purpose

Cane Corso Growth Geometry Lab is a focused web application that demonstrates machine learning concepts through a Cane Corso growth-orientation workflow.

The app is designed with two clear layers:

1. **Product layer** — a user can enter dog measurements and receive an orientation signal with formulas, maps and explanations.
2. **Project/methodology layer** — a reviewer can inspect the datasets, engineered features, model comparisons, PCA-style map, clustering logic, MLflow-style tracking plan and course-topic mapping.

## 2. Important boundary

The app provides orientation only. It does not prove health status, breed purity, pedigree or official Cane Corso status, and it does not replace professional veterinary advice.

## 3. Technologies

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Python notebooks
- pandas / numpy / scikit-learn foundation
- MLflow-ready experiment tracking concept

## 4. Machine learning topics shown

| Topic | How it appears in the app |
| --- | --- |
| Regression | Expected growth curve and estimated adult weight signal |
| Classification | Review-zone growth signal, not a diagnosis |
| Clustering | Growth profile groups based on feature distance |
| Feature Engineering | Raw user input is transformed into model-ready signals |
| Dimensionality Reduction | PCA-style map projects multiple features into 2D space |
| MLflow | Tracking concept for runs, metrics and artifacts |

## 5. User-facing flow

1. Open **Growth Check**.
2. Enter or load a demo Cane Corso profile.
3. Review the growth signal, coordinate map, formulas and explanation.
4. Inspect the methodology in **ML Methodology**, **Data & References** and **Project Evidence**.

## 6. Why this is both an app and a project

The visible interface avoids development wording such as patch, step and route names. The project evidence is still available in a dedicated methodology layer so the work remains suitable for course review.

## 7. Future extension

A future visual module can add photo guidance, photo-readiness checks, landmark geometry and visual Cane Corso similarity. That future module should use the same safety boundary: visual similarity is not proof of breed purity or pedigree.
