# Project Scope

## What this project is

**Cane Corso Growth Geometry Lab** is a separate experimental web app for exploring Cane Corso growth using coordinate-based thinking and machine learning concepts.

The project is intentionally built outside the main Cane Corso Platform repository. This keeps the experiment clean, focused and safe to evolve without affecting the production-oriented platform.

## Main concept

The central idea is simple:

- a dog profile becomes a point in a coordinate system;
- age can be placed on the X axis;
- weight or another growth metric can be placed on the Y axis;
- expected development can be represented as a curve;
- unusual values can be represented as review zones;
- similar profiles can later be grouped with clustering.

## What this project is not

This project is not:

- a veterinary diagnosis tool;
- a medical decision system;
- a replacement for a veterinarian;
- a production health platform;
- a certified breed evaluation system;
- a breeding recommendation engine.

## Safe wording

The app should use words such as:

- educational signal;
- orientation;
- review zone;
- growth pattern;
- model estimate;
- needs attention.

The app should avoid words such as:

- diagnosis;
- disease detection;
- guaranteed result;
- medical conclusion;
- certified health decision.

## Relationship to the main Cane Corso Platform

This app is separate from the main platform. It can later inspire a future module, but for now it should remain an independent ML experiment.
