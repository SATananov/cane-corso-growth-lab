# Data Policy

## Purpose

This project uses data only for educational machine learning experiments and prototype visualization.

## Data types

The project may use:

- synthetic sample data;
- public dataset samples;
- processed CSV files;
- manually created example profiles.

## Sensitive data rule

The project should not include private owner data, real contact information, addresses, veterinary documents or private dog medical records.

## Raw data rule

If a public dataset has licensing or redistribution restrictions, raw files should not be committed directly. Instead, the project should document where the data comes from and include only allowed processed samples.

## Veterinary safety rule

Data-based outputs should be described as educational model signals, not as medical results.
