# Step 16 — USG Lab Visual Lock

This step locks the visual direction of the Cane Corso Growth Geometry Lab as a USG-inspired experimental interface.

## Visual direction

The app should feel premium, dark, precise and evidence-focused:

- black graphite base
- gold used as an active signal color
- ivory/stone text hierarchy
- rounded premium cards
- glass/lab surfaces
- readable formulas, tables and model evidence
- clear experimental boundary

## Product boundary

The visual identity is intentionally USG-inspired, but the project remains a separate ML lab app. It should not look like an official registry, certificate or veterinary authority surface.

## UI surfaces affected

- global background and focus/scroll treatment
- navigation shell
- hero panels
- footer disclaimer block
- home hero
- calculator input card
- About visual system panel

## Why this matters

The project now has enough ML depth: regression, classification, clustering, feature engineering, PCA, MLflow, tables and formulas. This step makes the UI feel like one coherent product rather than many separate ML sections.
