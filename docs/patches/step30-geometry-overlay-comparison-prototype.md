# Step 30 — Geometry Overlay Comparison Prototype

Adds a comparison workspace that shows:

- Cane Corso reference geometry on the left;
- uploaded user photo on the right;
- photo gate permission state;
- blocked state for rejected photos;
- overlay comparison map;
- ratio evidence table;
- professional safety boundary.

This is not a trained vision model yet. It is the UI and logic bridge between the photo quality gate and the future landmark / visual similarity models.
