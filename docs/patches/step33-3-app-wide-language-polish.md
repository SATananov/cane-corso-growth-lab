# Step 33.3 — App-Wide Language Polish

This patch is a visible UI language pass after the visual ML modules were added. It reduces EN/BG/IT mixing in the main app modules while keeping standard ML terms such as Regression, Classification, PCA, MLflow, embedding and classifier where they are useful for project review.

Scope:
- Data page summary and feature engineering sections
- ML methodology page high-level workflow and model cards
- Research summary and figure gallery labels
- App model bridge labels
- Project evidence dashboard labels
- Visual review training/readiness panels

This patch does not change model formulas, datasets, image manifests, notebooks, Git remotes or package dependencies.
