# Small Curated Demo Image Set Plan

Step 32 prepares a controlled starter image set for the visual ML direction.

The goal is not to commit random images or claim a trained vision model. The goal is to define the first safe, manually reviewed image set that can later be used to demonstrate:

- photo readiness gate behavior;
- Cane Corso vs similar molosser visual classes;
- visual similarity comparison;
- geometry overlay examples.

## Minimum starter set

The first milestone is intentionally small:

| Group | Minimum | Target | Purpose |
|---|---:|---:|---|
| Cane Corso reference examples | 12 | 30 | Demo visual similarity and reference examples |
| Similar molosser examples | 12 | 30 | Hard-negative classifier examples |
| Unsuitable photos | 12 | 30 | Photo quality gate rejection/limited examples |

## Rules

Every image must have:

- source name;
- source URL or source note;
- license status;
- permission status;
- breed label or unknown label;
- view type;
- photo quality;
- comparison readiness;
- issues if limited or rejected.

Raw image folders are placeholders only. Do not commit real raw images unless their license and permission status explicitly allow repository inclusion.

## Visual language boundary

The demo set supports visual similarity only. It must not be used to claim breed purity, pedigree, genetic origin or official registration status.
