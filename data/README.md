# Data Folder

This folder contains the small research datasets used by the Cane Corso Growth Geometry Lab ML foundation.

## Included files

```txt
data/prototype/cane_corso_growth_sample.csv
data/processed/dog_growth_public_sample.csv
data/processed/dog_growth_classification_sample.csv
data/raw/source_notes.md
```

## Data use

The data is used only for educational machine learning experiments and app visualization.

The app must not treat these samples as veterinary authority, health diagnosis, official Cane Corso standard evaluation or breeding advice.
