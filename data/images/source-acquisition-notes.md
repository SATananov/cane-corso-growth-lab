# Image Source Acquisition Notes

This folder documents future image acquisition for the visual Cane Corso model.

Raw downloaded photos should not be committed to Git unless their license/permission explicitly allows that. Prefer local ignored folders or external storage for image files, and commit only source catalogs, manifests, labels and scripts.

The future visual model needs two kinds of evidence:

1. **Breed/type examples** — Cane Corso and visually similar comparison classes.
2. **Photo readiness examples** — accepted, limited and rejected photos that teach the gate which images can be compared to the reference geometry.
