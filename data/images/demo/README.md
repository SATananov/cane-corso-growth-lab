# Small Curated Demo Image Set

This folder is a placeholder for a future small, manually reviewed demo image set.

Raw images should not be committed unless their license and permission status allow repository inclusion. For normal work, keep raw images outside Git and commit only manifests, labels and documentation.

Recommended first milestone:

- 12 licensed Cane Corso reference images
- 12 licensed similar molosser breed images
- 12 unsuitable-photo examples for the quality gate

Each image must have a row in `data/images/labels/demo-image-manifest-template.csv` before it is used for a demo classifier or visual similarity experiment.
