# Notebooks

This folder is the research foundation for the web app.

## Notebook sequence

```txt
01_growth_regression_geometry.ipynb
02_real_data_preparation.ipynb
03_growth_classification_zones.ipynb
04_growth_clustering_experiment.ipynb
```

## Role in the app

The notebooks explain the ML thinking. The React app uses a simplified TypeScript bridge in `src/lib/ml` to present the same ideas in an interactive product form.

## Safety boundary

The notebooks are educational. They do not produce veterinary diagnosis, medical conclusions or certified breed decisions.
